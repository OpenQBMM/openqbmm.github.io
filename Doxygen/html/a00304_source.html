25 Application
26  Application to construct moments give other parameters. Builds initial
27  field files for main solver to use as input.
28 
29 Description
30  Solver for a system of any number of compressible fluid phases with a
31  common pressure, but otherwise separate properties. The type of phase model
32  is run time selectable and can optionally represent multiple species and
33  in-phase reactions. The phase system is also run time selectable and can
34  optionally represent different types of momentun, heat and mass transfer.
