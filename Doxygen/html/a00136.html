Detailed Description

template<class momentFieldSetType, class nodeType>
class Foam::quadratureApproximation< momentFieldSetType, nodeType >

Provides storage for the moments and the corresponding quadrature approximation of a univariate distribution function. Methods to update the moments and the quadrature approximation are provided.
