Detailed Description

Collision model which returns the velocity distribution to a Maxwellian distribution (multivariate Gaussian) with at a given collisional time scale. Used as a base class for esBGK as well, but a different covariance tensor is used as well a variable collision time scale. Polydispersity is also handled, but a variabel time scale is used.

Extension of the BGK collisional model that includes the coefficient of restitution.
