27 Description
28  Stores a list, with a Map<label> that takes a label key and returns
29  the index into the list.
30 
31  Example
32 
33  Lookup | Label Key | Index
34  0 0 0 0 0
35  1 0 0 100 1
36  0 1 0 10 2
37  0 0 1 1 3
38  2 0 0 200 4
39  2 1 0 210 5
40 
41  The operator () is overloaded in order to hide the above mapping.
