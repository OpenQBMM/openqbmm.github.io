27 Description
28  Calculates and applies the force necessary to maintain the specified mean
29  velocity.
30 
31  Note: Currently only handles kinematic pressure (incompressible solvers).

33  \heading Source usage
34  Example usage:
35  \verbatim
36  phaseCompressibleMeanVelocityForceCoeffs
37  {
38  selectionMode all; // Apply force to all cells
39  fieldNames (U.gas); //
40  alphaName alpha.gas;
41  rhoName rho.gas;
42  Ubar (10.0 0 0); // Desired mean velocity
43  relaxation 0.2; // Optional relaxation factor
44  }
